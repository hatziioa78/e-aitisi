# e-Aitisi (Σύστημα Ηλεκτρονικών Αιτήσεων)

Ένα ολοκληρωμένο, web-based σύστημα διαχείρισης ηλεκτρονικών αιτήσεων, σχεδιασμένο ειδικά για Διευθύνσεις Εκπαίδευσης (Δ.Δ.Ε. / Δ.Π.Ε.) και εκπαιδευτικούς οργανισμούς. Επιτρέπει στους εκπαιδευτικούς να υποβάλλουν αιτήσεις ηλεκτρονικά και στη Γραμματεία/Διαχειριστές να τις επεξεργάζονται, να αποδίδουν αριθμό πρωτοκόλλου και να εξάγουν αναφορές.

## ✨ Βασικά Χαρακτηριστικά

* **Ασφαλής Ταυτοποίηση (SSO):** Ενσωμάτωση με υπηρεσίες CAS (Central Authentication Service) του ΠΣΔ για την ασφαλή σύνδεση των εκπαιδευτικών και καταγραφή των στοιχείων τους (IP, Σχολείο, ΑΜ, κ.λπ.).
* **Δυναμικές Ρυθμίσεις (JSON Configuration):** Κεντρική διαχείριση ρυθμίσεων μέσω του `conf.json` με UI βασισμένο σε Drag & Drop (SortableJS) για εύκολη παραμετροποίηση χωρίς παρέμβαση στον κώδικα.
* **Αυτοματοποιημένο Setup Wizard:** Έξυπνος οδηγός εγκατάστασης (`setup.php`) που ελέγχει τα δικαιώματα δικτύου, συνδέεται στη βάση και δημιουργεί αυτόματα τη δομή των πινάκων (Automated Schema Migration) από αρχείο JSON.
* **Διαχείριση Πρωτοκόλλου:** Αυτόματη απόδοση και παρακολούθηση αριθμών πρωτοκόλλου.
* **Ειδοποιήσεις Email:** Ενσωματωμένο PHPMailer για την αποστολή επιβεβαιωτικών emails στη Γραμματεία και τους χρήστες.
* **Εξαγωγή σε Excel:** Χρήση της βιβλιοθήκης PhpSpreadsheet για την εξαγωγή δεδομένων και αναφορών.
* **Responsive UI:** Σύγχρονο και φιλικό περιβάλλον χρήστη χτισμένο με Bootstrap 5 και DataTables.

## 🛠️ Τεχνολογίες & Βιβλιοθήκες

* **Backend:** PHP 7.4 / 8.x
* **Database:** MySQL / MariaDB
* **Frontend:** HTML5, CSS3, JavaScript (jQuery)
* **Frameworks/UI:** Bootstrap 5, Bootstrap Icons
* **Plugins:** DataTables (με υποστήριξη Ελληνικών), SortableJS
* **3rd Party PHP Libs:** PHPMailer, PhpSpreadsheet
